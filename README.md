# Translator (Final Year Project)

A full-stack **Translator** web application that translates text between multiple languages using a frontend UI and a backend API.

## Features
- Translate text between supported languages
- Simple and responsive UI
- Backend API for translation requests

## Tech Stack
**Frontend:** [React / HTML-CSS-JS]  
**Backend:** [Python/Flask]  
**Database:** [Postgres]  

## Project Structure
- `frontend/` — Frontend client application
- `Backend/` — Backend server/API

## Installation & Setup

### 1) Frontend setup
```bash
cd frontend
npm install
npm run dev
```

### 2) Backend setup
```bash
cd Backend
pip install -r requirements.txt
py backapp.py
```
## Usage
1. Open the frontend in your browser.
2. Enter text to translate.
3. Select source and target languages.
4. Click **Translate** to view results.
